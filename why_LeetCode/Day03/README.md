题目1：Median of Two Sorted Arrays
   给定两个大小为 m 和 n 的有序数组nums1和nums2。

请你找出
   这两个有序数组的中位数，并且要求算法的时间复杂度为O(log(m+n))。
   

你可以假设nums1和nums2不会同时为空。



   nums1 = [1, 2]
   nums2 = [3, 4]

   则中位数是 (2 + 3)/2 = 2.5



题目2：Longest Consecutive Sequence
   给定一个未排序的整数数组，找出最长连续序列的长度。
   要求算法的时间复杂度为 O(n)。


   输入: [100, 4, 200, 1, 3, 2]
   输出: 4
   解释: 最长连续序列是 [1, 2, 3, 4]。它的长度为 4。
